Программа предназначена для выравнивания таблиц в текстовом файле (не Unicode):

Компиляция программы (с использованием стандартного компилятора C++ и STL библиотеки, например, g++ или clang++)

g++ -std=c++11 *.cpp -o parser

Запуск программы:

parser <входной_файл>

Результат работы программы будет сгенерирован в выходном файле с добавочным расширением ".out"

Пример запуска:

./parser test/tables.txt
